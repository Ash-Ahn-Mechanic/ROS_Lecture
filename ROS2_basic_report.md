로봇이라는 단어는 1920년 체코 작가 **카렐 차페크**의 연극에서 처음
등장했다.

# 4. 컴퓨터 부팅 과정

컴퓨터가 켜질 때 진행되는 과정

Power On\
→ POST 실행\
→ BIOS 설정 로드\
→ Boot Loader 실행\
→ OS Kernel 로드\
→ 초기 프로세스 실행\
→ 사용자 인터페이스 실행
