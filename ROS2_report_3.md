# 1. 전체 공통 핵심 내용

이 강의는 크게 다음 4가지 주제를 설명한다.

1. 센서와 로봇의 기본 구조
2. 로봇 시스템 구성
3. ROS2 개념
4. ROS2 설치 및 실습

로봇 시스템은 기본적으로 다음 구조로 동작한다.

입력 → 처리 → 출력

- 입력 : 센서
- 처리 : 컴퓨터 / 제어기
- 출력 : 액츄에이터

사람으로 비교하면 다음과 같다.

| 사람 | 로봇 |
|---|---|
눈, 귀, 피부 | 센서 |
뇌 | 컴퓨터 / 제어기 |
근육 | 액츄에이터 |
