# 1. 핵심 내용

## 1.1 ROS2 Bag이란
- ROS2 BAG은 시스템의 토픽에 게시된 데이터를 기록하기 위한 플랫폼 기능
- 각 토픽에서 수집된 데이터를 데이터베이스에 저장
- 저장된 데이터를 재생하여 테스트 및 실험 결과를 재현할 수 있음  
  <!-- 핵심은 "토픽 데이터 기록 + 재현" -->

## 1.2 왜 사용하는가
- 데이터 저장
- 실험 재현
- 디버깅
- 센서 데이터 재생  
  <!-- 실제 장비 없이도 이전 데이터를 다시 써서 테스트 가능 -->

## 1.3 기본 명령어

### 레코딩
```bash
ros2 bag record -o my_bag /topic_name
```
<!-- 특정 토픽을 my_bag 이름으로 기록 -->

### 재생
```bash
ros2 bag play my_bag
```
<!-- 저장된 bag 파일 재생 -->

### 정보 확인
```bash
ros2 bag info my_bag
```
<!-- bag 파일 안의 토픽, duration, storage 정보 확인 -->

# 2. 핵심 실습 1 — Turtlesim Bag 파일 만들기

## 2.1 실습 목표
- Turtle의 움직임을 BAG 파일에 기록
- 기록된 BAG 파일을 다시 재생하여 동일한 움직임을 재현  
  <!-- 가장 기본적인 bag 사용 실습 -->

## 2.2 준비 단계
- 여러 터미널을 사용하여 진행
- 터미널 3개를 미리 열어둘 것을 권장  
  <!-- turtlesim / teleop / record 를 따로 실행하기 때문 -->

## 2.3 1단계 — 첫 번째 터미널
```bash
ros2 run turtlesim turtlesim_node
```
<!-- turtlesim 실행 -->

## 2.4 2단계 — 두 번째 터미널
```bash
ros2 run turtlesim turtle_teleop_key
```
<!-- 키보드로 turtle 조작 -->

## 2.5 3단계 — 세 번째 터미널에서 기록 시작
```bash
ros2 bag record -o turtle_bag /turtle1/cmd_vel
```
- `/turtle1/cmd_vel` 토픽을 `turtle_bag` 이름의 BAG 파일에 기록
- 반드시 `Subscribed to topic '/turtle1/cmd_vel'` 문구가 뜨는지 확인  
  <!-- 속도 명령 토픽을 저장 -->

## 2.6 4단계 — 실제 움직임 기록
- 두 번째 터미널로 이동
- 방향키를 사용해 turtle을 임의로 이동시킴  
  <!-- bag에 실제 움직임 데이터가 쌓이는 단계 -->

## 2.7 5단계 — 기록 종료
- 세 번째 터미널에서 `Ctrl + C`로 record 종료  
  <!-- 이 시점에 bag 파일이 저장됨 -->

## 2.8 6단계 — 저장된 파일 확인
```bash
ros2 bag info turtle_bag
```
<!-- 저장된 bag 파일의 기본 정보 확인 -->

## 2.9 7단계 — 재생 준비
- 첫 번째 터미널에서 실행 중인 turtlesim 종료 후 재실행  
  <!-- 기록 당시와 비슷한 초기 상태로 맞춤 -->

## 2.10 8단계 — 재생
```bash
ros2 bag play turtle_bag
```
- 저장된 BAG 파일 재생
- 간혹 인터럽트 등의 이유로 궤적이 정확히 일치하지 않을 수 있음
- 이를 줄이려면 기록할 때 각 명령이 완전히 완료된 후 다음 명령을 실행해야 함  
  <!-- 실습상 주의사항 -->

## 3.1 주제

- 3D 카메라
- LiDAR를 이용하여 기록된 BAG 파일 실행  
  <!-- turtlesim 이후 실제 센서 데이터 활용으로 확장 -->

## 3.2 2D 카메라 Bag 파일 실습
- 제공된 링크에서 `rosbag2_video.tar.gz` 다운로드
- 압축 해제 후 bag 파일 정보 확인
- bag 파일 반복 재생
- 새로운 터미널에서 `rviz` 실행  
  <!-- 영상 데이터를 rviz에서 시각적으로 확인 -->

## 3.3 RViz 연동
- RViz에서 `Add` 버튼 클릭
- 카메라 관련 디스플레이 추가
- 재생 중인 영상 토픽을 연결하여 화면에 표시  
  <!-- Bag 데이터와 시각화 도구를 연결하는 핵심 단계 -->

## 3.4 실습 의미
- bag은 단순 저장 파일이 아니라
- 카메라 / LiDAR 같은 센서 데이터를 다시 불러와 시각화하고 분석할 수 있는 재현 도구

# 5. 실습 흐름 요약

## 5.1 Turtlesim Bag 흐름
```text
turtlesim 실행
-> teleop 실행
-> /turtle1/cmd_vel 기록
-> turtle 이동
-> record 종료
-> bag info 확인
-> bag play 재생
```
<!-- 가장 기본적인 기록-재생 실습 -->

## 5.2 센서 Bag 흐름
```text
bag 파일 준비
-> bag info 확인
-> bag play 실행
-> rviz 실행
-> topic 연결
-> 영상 / 센서 데이터 확인
```

# 6. 핵심 정리

- ROS2 Bag은 토픽 데이터를 기록하고 재생하는 기능
- 실험 결과를 다시 재현할 수 있음
- turtlesim처럼 단순한 제어 데이터도 저장 가능
- 카메라 / LiDAR처럼 실제 센서 데이터도 저장 및 시각화 가능
- RViz와 함께 사용하면 재생 데이터를 눈으로 확인할 수 있음
