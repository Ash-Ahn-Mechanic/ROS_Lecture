## 4. 리눅스 파일 시스템 구조

리눅스는 트리 구조로 파일을 관리한다.

가장 위 폴더

    /

이를 **루트 디렉토리**라고 한다.

예시 구조

    /
     ├ bin
     ├ etc
     ├ home
     ├ usr
     ├ tmp
     └ var

각 폴더 의미

  폴더    역할
  ------- -------------
  /bin    기본 명령어
  /boot   부팅 파일
  /dev    장치
  /etc    설정
  /home   사용자 폴더
  /usr    프로그램
  /var    로그 데이터

이 구조를 **FHS(File System Hierarchy Standard)** 라고 한다.

## 15. 독창적인 내용 : Y2038 문제

32bit 시스템은 다음 시간까지 표현 가능

    2,147,483,647 초

이 시간이 지나면

    2038년 1월 19일

시간이 overflow 되어 과거 시간으로 돌아가는 문제가 발생한다.

해결

1.  unsigned int 사용
2.  64bit time 사용

64bit 시간은 약 **2922억년까지 사용 가능**하다.
